---

## 🛠️ Tech Stack
- **Framework:** Next.js  
- **Styling:** Tailwind CSS  
- **Icons:** Lucide React  
- **Theme:** Custom dark/light provider

---

## 🚀 Run Locally

```bash
git clone https://github.com/vibestudent/vibe-portfolio.git
cd vibe-portfolio
npm install
npm run dev
```

Open: http://localhost:3000

(If you prefer yarn: replace `npm` with `yarn`.)
